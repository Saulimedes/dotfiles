# Dotfiles
## About
This repo is a collection of all my dotfiles managed by dotbot.

## Installation
```
git clone https://gitlab.com/paulbecker/dotfiles .dotfiles
cd .dotfiles
./bootstrap
```

## What gets installed
* Distro: opensuse
* Bootstrap Installation
* Editor: emacs
* Fonts: PramataPro
* fish with plugins managed by fisher
* Git Helper
