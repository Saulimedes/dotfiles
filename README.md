## About

This repo is a collection of all my dotfiles managed by Ansible.

## Installation

```shell
git clone git@github.com:Saulimedes/dotfiles.git .dotfiles
cd .dotfiles
# install and configure system
./bootstrap

# just configure system
./config
```
