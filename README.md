## About
This repo is a collection of all my dotfiles managed by dotbot.

## Installation
```
git clone git@github.com:Saulimedes/dotfiles.git .dotfiles
cd .dotfiles
./bootstrap
```

## What gets installed
* Distro: opensuse
* Bootstrap Installation
* Editor: neovim
* Fonts: MonoLisa
* fish with plugins managed by fisher
* Git Helper
